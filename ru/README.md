# Панель администрирования маршрутизатора TP-Link

![Github repo license](https://img.shields.io/github/license/KagamineP/tplrap) ![Github repo size](https://img.shields.io/github/languages/code-size/KagamineP/tplrap)

Эта программа была создана для упрощения входа в административную панель на маршрутизаторах [TP-Link](https://tp-link.com/) . Эта программа требует подключения к маршрутизатору через Wi-Fi или локальную сеть. Эта программа использует [CefSharp](https://github.com/cefsharp/CefSharp) , поэтому программа загружает страницу http://tplinkwifi.net/ (работает только на маршрутизаторах TP-Link). Т.е. эта программа является ярлыком для доступа к панели администрирования.

Функция автообновления сделана на основе [AutoUpdater.NET](https://github.com/ravibpatel/AutoUpdater.NET) .

## Известные вопросы

Нет возможности доступа к панели администратора на старых версиях роутеров. Временное решение: используйте любой браузер, например: Internet Explorer 11.

## Системные Требования

Операционная система: Windows 7 и выше.
 .NET Framework 4.7.2

## Зависимости

Для того, чтобы собрать данную программу, Вам нужно восстановить пакеты NuGet.
